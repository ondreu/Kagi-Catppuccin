# Kagi-Catppuccin
Vibe coded Kagi custom CSS using Catppuccin palete, predominantly mauve.
I vibe coded this Catppuccin flavor becouse i wasnt happy with the, at this time [only other Catppuccin CSS](https://github.com/jcrabapple/kagi-catppuccin-light-dark/tree/main) I found.

This Catppuccin theme is focused on beaing softer, less neon, and more professional.

## To instal
1. Open [raw file](https://raw.githubusercontent.com/ondreu/Kagi-Catppuccin/refs/heads/main/Custom-CSS)
2. CTRL+A and CTRL+C to copy all contents.
3. Go to Kagi setting -> Appearance -> Custom CSS|Change
4. Paste the copied text into the window and save changes.
5. Toogle Custom CSS switch.

---
To change font from sans-serif to monospace change this:
```
/* === FONT === */

body { font-family: sans-serif; }
```
to this:
```
/* === FONT === */

body { font-family: monospace; }
```
